CARA BUAT MARQUEE

1. Ini adalah contoh pertama marquee paling asas, secara default kearah kiri:

Code / Syntax:

<marquee>Teks bergerak</marquee>

Result / Output:

Teks Bergerak

2. Marquee kearah kanan:

Code / Syntax:

<marquee direction="right">Teks bergerak</marquee>

Result / Output:

Teks Bergerak

3. Sekiranya, mahu menarik kita tambah dengan style warna latar & warna tulisan:

Code / Syntax:

<marquee style="background-color: #000000; color: #ffffff">Teks bergerak</marquee>

Result / Output:

Teks Bergerak

4. Tulisan bergerak dari kanan ke kiri dan berpatah balik ke kanan:

Code / Syntax:

<marquee behavior="alternate">Teks bergerak</marquee>

Result / Output:

Teks Bergerak

5. Tulisan bergerak dari kiri ke kanan dengan scrollamount 10 lebih laju berbanding biasa:

Code / Syntax:

<marquee direction="right" scrollamount="10">Teks bergerak</marquee>

Result / Output:

Teks Bergerak
